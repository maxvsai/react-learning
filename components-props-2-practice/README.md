# Components & Props - Lesson 2 Practice

Build a product catalog using composable components, `children`, default props, and the spread operator.

## Tasks

1. Create a `Card` component with `title`, `children`, and optional `footer`
2. Create a `Badge` component with default prop values
3. Create a `ProductCard` that composes Card + Badge
4. Render a list of products using the spread operator

## Run

```bash
npm install
npm run dev
```

When done, run `/submit components-props` in Discord!
